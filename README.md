# electron-doc

#### 介绍
我的文档桌面应用
实现文档新建、导入、查找、编辑、删除等功能，实现数据持久化保存，自定义窗口菜单，可更改文档保存路径，实现应用自动更新版本。

#### 软件架构
electron reactHooks node bootstrap

#### 安装教程

1.  npm install
2.  npm run dev（ 启动）
3.  npm run dist (打包）
